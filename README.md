Watch_Pypilot

A modern ESP32-C6 touchscreen remote control for pypilot.

Watch_Pypilot turns a Waveshare ESP32-C6 2.06" AMOLED smartwatch into a fully featured wireless autopilot remote with a simple, marine-oriented interface.

The watch communicates directly with a pypilot server over Wi-Fi and allows full control of the autopilot while displaying essential navigation information.
